# ROS2 Fundamentals Examples

This package contains examples demonstrating fundamental ROS 2 concepts.

## Description

This package includes minimalist ROS 2  code to demonstrate important ROS 2 concepts and patterns.
- Following ROS 2 Python style guidelines

## Prequisites

- ROS 2 installed
- Python 3
- Created a ROS 2 workspace ('ros_2ws')
